**Архитектурные изменения в `index.html`**:

- Легаси `cm()` **нетронута** — все BUG-FIX'ы (BUG2, BUG5, BUG12) сохранены

- `renderScenarios.sc()` и `renderSensitivity.d()` применяют коэффициенты `pm/sm/skuPrice/skuPurchase/skuLoss` ко всем SKU одновременно
- `A.mosh` заменён на `(A.trucks_per_year*A.truck_kg)` во всех render'ах (mosh оставлен в A_DEFAULT для совместимости с легаси cm)

- Новый блок «Ассортимент — что везём в фуре»: 3 кнопки пресетов, кнопка `+ SKU` (лимит 8), таблица SKU с инлайн-полями (название, цена тг/шт, закупка €/кг, вес г, потери %, доля), `×` для удаления
